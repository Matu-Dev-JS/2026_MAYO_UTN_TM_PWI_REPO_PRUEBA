## Para empezar un nuevo proyecto

1. git init -> Te crea el repositorio en el directorio actual

2. git add . -> Añadir los archivos al repositorio

3. git commit -m 'mensaje descriptivo del cambio' -> Versionar el cambio

4. git branch -M main -> Renombra la rama master a main (OPCIONAL)

5. git remote add origin https://github.com/Matu-Dev-JS/2026_MAYO_UTN_TM_PWI_REPO_PRUEBA.git -> Añadir un origen remoto (Donde vamos a subir el codigo)

6. git push -u origin main -> Subis tu commit mas actual al repositorio remoto

## Para actualizar el repositorio 

1. git add . -> Añadir los archivos al repositorio

2. git commit -m 'mensaje descriptivo del cambio' -> Versionar el cambio

3. git push -> Subis tu commit mas actual al repositorio remoto
